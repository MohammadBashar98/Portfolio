Overview

Pro Unity Game Development with C# by Alan Thorn is a well-regarded game development textbook, but its code targets a much older version of Unity. Rather than follow along with outdated APIs and deprecated workflows, I rebuilt the featured project — Crazy Mad Office Dude — entirely in Unity 6, adapting every system to current standards along the way.

The game is a top-down shooter where the player navigates office-themed stages, collecting all the money pickups to advance. Enemies patrol the environment and the player must manage movement, shooting, and pickups while clearing each level. The focus of the project was to work through the book's design decisions critically, understanding why each system was built the way it was and how those decisions translate — or don't — into Unity 6.

Features
  • Top-down shooter gameplay with money-collection win condition per stage
  • Full rebuild in Unity 6 — all deprecated APIs replaced with current equivalents
  • Based on Alan Thorn's Pro Unity Game Development with C#, adapted and modernised throughout
  • Enemy patrol and player combat systems implemented from scratch in C#
  • Stage progression flow with pickup tracking and level-clear logic
